:::note टेर
द्वार खड़े भोले नाथ मैं तो जाउंगी सत्संग में 
:::

- सास मेरी जाय रही ननद मेरी जाय रही <br/>
मैं भी लग लई साथ मैं तो जाउंगी … <br/>
द्वार खड़े ..... 				|| १ || 

- सास लाइ ढोलक ननद लाइ चिमटा <br/>
हम लाए खड़ताल मैं तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| २  || 

- सास की फूटी ढोलक ननद को टूटो चिमटो <br/>
खूब बजाई खड़ताल मैं तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| ३  || 

- जेठ मेरे जाय रहे जेठानी मेरी जाय रही <br/>
हम हूँ लग लये साथ में तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| ४ || 

- जेठ लाये लड्डू जेठानी लाये पेड़ा <br/>
हम लाए प्रसाद मैं तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| ५ || 

- फूट गए लड्डू बिखर गए पेड़े <br/>
खूब बटो प्रसाद मैं तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| ६ || 

- देवर मेरे जाय रहे देवरानी मेरी जाय रही <br/>
हम हूँ चल गये साथ मैं तो जाउंगी। … <br/>
द्वार खड़े ..... 				|| ७ || 

- देवर लाये हांडी देराणी लाइ मटका <br/>
हम लाये गंगाजल हम तो जाएंगे सत्संग में <br/>
द्वार खड़े ..... 				|| ८ || 

- फूट गयो मटको ने फूट गई हांडी <br/>
खूब नहाये भोले नाथ मैं तो। … <br/>
द्वार खड़े ..... 				|| ९ ||
